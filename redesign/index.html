OHI/O is a program dedicated to fostering tech culture at The Ohio State University.

The student-driven program has created Ohio State's first hackathon, as well as spin-off events including the hardware-focused Makeathon, ultimately providing students the opportunity to learn and build with real technologies outside of the classroom.
